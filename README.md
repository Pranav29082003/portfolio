# Data Analyst Portfolio Website

A clean, modern, responsive personal portfolio website designed to showcase data analytics case studies, technical skills, and industry certifications.

## 🚀 Features
Interactive Hero Section: Features a dynamic typing text effect driven by js/typing.js highlighting key capabilities.

Data-Centric Design: Built using a customized, clean dashboard-like color palette to match data engineering and analytical theme requirements.

Dynamic Project Injection: Managed dynamically via js/projects.js to scale features easily when launching new data case studies.

Fully Responsive Layout: Designed mobile-first using flexible grids and specialized breakdown behaviors in css/responsive.css.

Structured Case Studies: Set up to display data collection, cleaning methodologies (SQL/Python), interactive visuals (Tableau/Power BI), and business impact.

## 🛠️ Technology Stack Used
Front-End Architecture: HTML5, CSS3 (Flexbox & Native CSS Grids), Modular JavaScript (ES6+).

Analytical Theme Tooling (Asset Placeholders): Python (Pandas/NumPy), SQL, Power BI, Tableau, Advanced Excel.


## 🔧 Getting Started & Installation
To run this portfolio locally on your machine, follow these steps:

Clone the Repository:

Bash
git clone [https://github.com/yourusername/portfolio.git](https://github.com/yourusername/portfolio.git)
Navigate into the project root directory:

Bash
cd portfolio
Launch the site:

Double-click index.html to open it in any modern browser.

Alternatively, if using Visual Studio Code, right-click index.html and select "Open with Live Server" to view real-time styling updates.
